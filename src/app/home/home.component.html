CGPA Calculator for Anna University Students Regulation - 2017

Here comes the solution for the students of anna university to solve the problem of calculating gpa and cgpa. Now it is easy to create their own profile and save the data of all the semesters within the website itself and can be accessed at any time.

Features of CGPA-Calculator

Access Semester Records

An Additional Feature of access to semester records is introduced in the application where each user's gpa record is stored in the dashboard of the registered students.

CGPA to Percentage

CGPA to Percentage is a additional feature which is added in the application that converts the cgpa to percentage.

Free Access and User Friendly

CGPA calculator is an application which is user friendly and helps students to access free and is a open source resource.
